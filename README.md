# 🏡 Airbnb Clone Website

A full-stack Airbnb-style web application where users can explore, list, and manage rental properties.

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 🏡 Create, Edit & Delete Listings
- 📍 Location-based property listing
- 🖼️ Image upload for properties
- 💬 Reviews & Ratings system
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

### Frontend
- HTML, CSS, JavaScript
- Bootstrap / Tailwind (based on your UI)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)
